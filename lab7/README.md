# Лабораторна робота №7: Computer Vision та багатопотоковість у C++

## Опис

Програма написана мовою C++ з використанням бібліотеки OpenCV.

Програма:
- отримує відео з камери;
- відображає відео у вікні;
- реагує на натискання клавіш;
- виконує різні режими обробки зображення;
- виконує детекцію обличчя за допомогою нейронної мережі;
- має інтерактивні елементи керування.

---

## Інструкція зі збірки та запуску

Надати права на виконання скриптів:

```bash
chmod +x preinstall.sh build.sh run.sh
```

Встановити залежності та завантажити файли нейронної мережі:

```bash
./preinstall.sh
```

Зібрати проєкт:

```bash
./build.sh
```

Запустити програму:

```bash
./run.sh
```

## Архітектура

### CameraProvider

Відповідає за роботу з камерою.  
Містить `cv::VideoCapture` і метод `getFrame()`, який повертає кадр з камери.

### KeyProcessor

Обробляє натискання клавіш.  
Містить `enum class Mode`, у якому зберігаються режими обробки зображення.

### FrameProcessor

Виконує обробку кадру залежно від вибраного режиму: інверсія, grayscale, Gaussian blur, Canny, Sobel, threshold.

### FaceDetector

Виконує детекцію обличчя за допомогою OpenCV DNN та моделі ResNet-10.

Для рівня 2 детекція винесена в окремий потік:
- `std::thread` використовується для фонового виконання нейромережі;
- `std::mutex` захищає спільні дані між потоками;
- `std::atomic<bool>` використовується для безпечної зупинки фонового потоку.

Основний потік читає кадри з камери та відображає відео, а фоновий потік виконує `net.forward()` і оновлює координати знайдених облич.

### Display

Відповідає за відображення результату у вікні через `cv::imshow`.

### main.cpp

Створює всі об’єкти класів і містить основний цикл програми.
