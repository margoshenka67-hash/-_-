## Опис 
В цій роботі потрібно було розробити веб-додаток, використовуючи бібліотеку Streamlit для аналізу часових рядів індексів VCI, TCI та VHI, 
отриманих у другій лабораторній. Основною метою є створення інтерактивного інструменту, який дозволяє користувачу виконувати фільтрацію даних за областями, 
роками та тижнями, а також обирати показник для аналізу.
У додатку реалізовано набір інтерактивних елементів (dropdown-списки, повзунки, checkbox та кнопка скидання), 
які забезпечують гнучке налаштування вибірки даних. 
Результати відображаються у вигляді таблиці та графіків: часових рядів для обраної області та порівняльного аналізу між областями.

## Структура папки проєкту
* `lab5.py` — основний файл завдання.
* `requirements.txt` — список необхідних бібліотек.
* `.gitignore` — виключення великих датасетів та системних файлів.

## Вимоги до системи
* **Інтерпретатор:** Python 3.9+
* **Необхідні бібліотеки:** `pandas`, `streamlit`, `plotly`.
