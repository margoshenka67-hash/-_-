# Лабораторна робота №6: OpenCV C++

## Опис

Програма написана мовою C++ з використанням бібліотеки OpenCV.

Програма:
- отримує відео з камери;
- відображає відео у вікні;
- реагує на натискання клавіш;
- виконує різні режими обробки зображення;
- має інтерактивні елементи керування.

Програма має модульну архітектуру (кожен модуль - окремий клас в окремих файлах .hpp та .cpp)

## Інструкція зі встановлення та запуску

### 1. Надати права на виконання скриптів

```bash
chmod +x preinstall.sh build.sh run.sh
```

### 2. Встановити залежності

```bash
./preinstall.sh
```

### 3. Зібрати проєкт

```bash
./build.sh
```

### 4. Запустити програму

```bash
./run.sh
```

## Реалізований функціонал

### Режими обробки зображення

- звичайне відео;
- інверсія кольорів;
- відтінки сірого;
- Gaussian blur;
- Canny edge detector;
- Sobel filter;
- бінаризація зображення.

---

## Керування

| Клавіша | Дія |
|---|---|
| 0 | Звичайний режим |
| 1 | Інверсія кольорів |
| 2 | Grayscale |
| 3 | Gaussian blur |
| 4 | Canny edge detector |
| 5 | Sobel filter |
| 6 | Threshold |
| q | Вихід з програми |

---

## Інтерактивні елементи

У програмі реалізовано:
- слайдер зміни яскравості;
- слайдер зміни порогу Canny edge detector;
- відображення FPS;
- відображення поточного режиму роботи.

## Архітектура програми

### CameraProvider
Клас для роботи з камерою через `cv::VideoCapture`.

### KeyProcessor
Клас для обробки натискань клавіш та зміни режимів роботи.

### FrameProcessor
Клас для виконання обробки зображення залежно від вибраного режиму.

### Display
Клас для відображення кадру у вікні через `cv::imshow`.

### main.cpp
Містить головний цикл програми та логіку взаємодії між класами.
